# Chronos

> [!tldr]
> CLI for managing chron jobs on a system. Chron jobs can be viewed and managed, and one-off jobs can be set.  

## Features
- View and manage chron jobs on a system in a table format
   - Quickly search for chron jobs by task
- Schedule one-off chron jobs for actions to be taken place at a certain time 


## Usages
- Have Claude schedule a command when the rate limit is reset
- Schedule a chron job to fire claude to write a daily brief every morning
- See what chron jobs have been set.
